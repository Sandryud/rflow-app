# Модуль `users`

## Назначение

Модуль отвечает за текущего аутентифицированного пользователя.

Его основная роль:

- возвращать профиль пользователя по JWT-идентификатору;
- предоставлять безопасный контекст текущего пользователя для остальных модулей;
- не хранить и не менять организационный или проектный контекст, а только данные учетной записи.

Подробные сценарии HTTP-операций описаны в отдельных endpoint-документах в [src/modules/users/docs/endpoints](./endpoints).
