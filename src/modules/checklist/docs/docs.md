# Модуль `checklist`

## Назначение

Модуль отвечает за checklist релиза.

Его основная роль:

- хранить список обязательных и необязательных пунктов для релиза;
- возвращать чек-лист релиза по его идентификатору;
- создавать новые пункты, привязывать их к релизу и пользователю-исполнителю;
- контролировать переходы статусов пунктов в рамках допустимого жизненного цикла.

Модуль работает в контексте релиза и организации и не описывает сам процесс публикации релиза.

Подробные сценарии HTTP-операций описаны в отдельных endpoint-документах в [src/modules/checklist/docs/endpoints](./endpoints).
