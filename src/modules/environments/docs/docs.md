# Модуль `environments`

## Назначение

Модуль отвечает за окружения проекта.

Его основная роль:

- возвращать список окружений для конкретного проекта;
- создавать новые окружения внутри проекта при наличии прав на это действие;
- хранить конфигурационный контекст, который затем используется в сценариях релизов и деплоя.

Модуль работает в контексте проекта и организации и не описывает сами релизы или их запуск.

Подробные сценарии HTTP-операций описаны в отдельных endpoint-документах в [src/modules/environments/docs/endpoints](./endpoints).
