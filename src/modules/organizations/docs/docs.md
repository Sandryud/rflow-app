# Модуль `organizations`

## Назначение

Модуль управляет организациями, доступными текущему пользователю.

Его основная роль:

- хранить и возвращать организации, в которых состоит пользователь;
- создавать организацию и назначать ее создателя владельцем (`OWNER`);
- обеспечивать доменный контекст для дальнейшей работы с участниками, проектами и релизами.

Подробные сценарии HTTP-операций описаны в отдельных endpoint-документах в [src/modules/organizations/docs/endpoints](./endpoints).
