# Модуль `auth`

## Назначение

Модуль отвечает за аутентификацию пользователя и создание учетной записи.

Основная роль модуля:

- выполнять вход пользователя по email и password;
- создавать нового пользователя с безопасным хешированием пароля;
- выдавать JWT access token для последующих защищенных запросов.

Модуль не содержит доменной бизнес-логики организации или проекта и не управляет контекстом участия пользователя в рабочем пространстве.

Подробные сценарии HTTP-операций описаны в отдельных endpoint-документах в [src/modules/auth/docs/endpoints](./endpoints).
