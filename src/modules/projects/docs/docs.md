# Модуль `projects`

## Назначение

Модуль отвечает за проекты внутри организации.

Его основная роль:

- возвращать список проектов по организации, в которой пользователь состоит;
- создавать новый проект в организации при наличии прав на это действие;
- хранить базовый проектный контекст и привязывать проект к конкретной организации.

Модуль работает поверх организационной модели и не управляет релизами, проверками или участниками проекта напрямую.

Подробные сценарии HTTP-операций описаны в отдельных endpoint-документах в [src/modules/projects/docs/endpoints](./endpoints).
